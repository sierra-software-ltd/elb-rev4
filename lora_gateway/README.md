RFM95 LoRa on Pocket Beagle board
=================================

Hardware setup
--------------

|Pocket beagle | RFM95       |
|--------------|-------------|
| P2.MISO      | MISO        |
| P2.MOSI      | MOSI        |
| P2.SCK       | SCK         |
| P2.CS        | NSS         |
| P2.45        | RESET       |
| P1.26        | DIO0        |

SPIDEV device is /dev/spidev2.1
SPI clock = 8MHz or 10MHz
